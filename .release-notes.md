### 🚀 대규모 고급 기능 업데이트 (v0.6.0)

선택하신 세 가지 핵심 기능이 모두 추가되었습니다!

*   **📊 메모리 사용량 시각화 (Memory Usage Inspector):** 컴파일(Verify)을 완료하면 대시보드 하단에 Flash와 RAM 점유율이 시각적인 프로그레스 바(Progress Bar)로 렌더링됩니다.
*   **📦 프로젝트 종속성 관리 (Dependency Management):** 작업 폴더 최상단에 `arduino.json` 파일 내부의 `"libraries": ["Servo", "DHT sensor library"]` 배열을 탐지하여, 누락된 라이브러리가 존재할 경우 원클릭으로 일괄 자동 설치합니다.
*   **🐛 시리얼 모니터 고급화 (Enhanced Serial Monitor):**
    *   **Timestamp (시간 표시):** 설정에서 `arduino.serialTimestamps`를 활성화하면 시리얼 데이터 수신 시 맨 앞에 로컬 타임스탬프가 추가됩니다.
    *   **Hex View (16진수 뷰):** 설정에서 `arduino.serialHexView`를 활성화하면 데이터를 16진수 바이트열로 표시합니다.
    *   **Log Export (로그 저장):** 커맨드 팔레트(`F1`)에서 `Arduino: Export Serial Log` 명령어를 실행하면 최근 5,000줄 분량의 로그를 텍스트 파일(.txt/.csv)로 선택된 위치에 저장합니다.
